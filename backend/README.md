### Getting Started

1. Clone Repo

```bash
git clone https://github.com/park-e/Parke-Backend.git
```

2. Change Directory

```bash
cd Backend
```

3. Install Dependencies

```bash
npm install
```

4. Add Environment Variables

```bash
touch .env

Add MONGO_URI, SECRET_KEY, PORT , CLIENTID , SENDINBLUEPASS
```

5. Start Server

```bash
npm start
```
